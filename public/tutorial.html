To get started

• Sign up at backpack to get the keys and cert

• Install the cert to your Trusted Root CA

• Please use basic auth with a key as username and your password as password (either key is fine for help message)

Examples

Demo of Geo query

Add locations to the set of "washingtoncities":

PUT https://backpack.ddns.net/store/set/washingtoncities/places

PUT BODY: [{"name":"vancouver","loc":{"type": "Point", "coordinates": [ -123.1, 49.25 ] }}, ...]


Query all locations in the set of "washingtoncities":

GET https://backpack.ddns.net/store/set/washingtoncities/places

Result: [{"_id":"54a34de1773ba0ba41111180","name":"vancouver","loc":{"type":"Point","coordinates":[-123.1,49.25]}},{"_id":"54a34de1773ba0ba41111181","name":"tacoma","loc":{"type":"Point","coordinates":[-122.4594,47.2414]}},{"_id":"54a34de1773ba0ba41111182","name":"kirkland","loc":{"type":"Point","coordinates":[-122.1917,47.6858]}},{"_id":"54a34de1773ba0ba41111183","name":"redmond","loc":{"type":"Point","coordinates":[-122.1239,47.6694]}}]


Query locations within 10km of bellevue ([ -122.1667, 47.6 ]):

POST https://backpack.ddns.net/store/set/washingtoncities/places

POST BODY: {"loc":{"$near":{"$geometry":{"type": "Point", "coordinates": [ -122.1667, 47.6 ] }, "$maxDistance":10000}}}

Result: [{"_id":"54a34de1773ba0ba41111182","name":"kirkland","loc":{"type":"Point","coordinates":[-122.1917,47.6858]}},{"_id":"54a34de1773ba0ba41111183","name":"redmond","loc":{"type":"Point","coordinates":[-122.1239,47.6694]}}]


Demo of Regex query

Query location with name starting with "red":

POST https://backpack.ddns.net/store/set/washingtoncities/places

POST BODY: {"name": {"$regex": "^red.*"}} Result: [{"_id":"54a34de1773ba0ba41111183","name":"redmond","loc":{"type":"Point","coordinates":[-122.1239,47.6694]}}]
